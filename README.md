# Free_proxy_pool
对免费代理IP网站进行爬取，收集汇总为自己的代理池。</br>
关键是验证代理的有效性、匿名性、去重复。</br>

<p>
本代理池的定位是初学者能看懂，能使用的单机库。所以不打算使用高大上的Redis或者MongoDB等数据库。抓到的代理仅与磁盘文件交互。</br>
最终会提供简洁易用的API，如get_a_proxy()，便于在请求网页的参数中直接使用。</br>
</p>

如果感觉对您有帮助，欢迎给我加一个星星，或者fork。
